# Day_27-Miles-to-Km-convertor-project
This simple project was built using Python's Tkinter library. This project allows users to input a distance in miles, click a button, and instantly see the converted value in kilometers.

## How the code works:
- A window is created using tkinter
- Input is taken through an Entry field where the user types the number of miles
- When the “Calculate” button is clicked the input value is retrieved
- It is converted into kilometers using a formula
- The result is displayed dynamically on the screen
- Labels are used to guide the user (e.g., “Miles”, “is equal to”, “Km”)
- The layout is managed using the grid() system to neatly organize components

## What I learned:
- Creating GUI applications with Tkinter
- Creating windows using Tk()
- Adding and configuring labels, buttons, and input fields
- Working with UI components:
    Button → triggers functions
    Entry → captures user input
- Updating UI elements dynamically using .config()
- Using grid() to structure elements in rows and columns
- Understanding alternative layout managers like:
    pack()
    place()
- *args → handling multiple positional arguments
- **kwargs → handling multiple keyword arguments
- Setting default values in function parameters
- Running functions based on user actions (e.g., button clicks)


This was a simple but powerful introduction to building interacting GUI applictaions. The project introduced the foundations of GUI development while also reinforcing flexible function design using *args and **kwargs.
It also gave me a glimpse into how real-world apps come to life.
